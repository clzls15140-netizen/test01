# 마크다운(Markdown) 사용법 총정리

마크다운은 간단한 기호를 사용해 텍스트에 서식을 지정할 수 있는 가벼운 마크업 언어입니다.

---

## 1. 제목 (Headers)
`#`의 개수로 제목의 크기를 결정합니다. (1개부터 6개까지 가능, `#` 뒤에 한 칸 공백 필수)

# 가장 큰 제목 (h1)
## 두 번째 제목 (h2)
### 세 번째 제목 (h3)
#### 네 번째 제목 (h4)
##### 다섯 번째 제목 (h5)
###### 가장 작은 제목 (h6)

---

## 2. 텍스트 스타일 (Text Styling)
글자에 굵기, 기울임, 취소선 등의 효과를 줄 수 있습니다.

*이 텍스트는 기울임꼴(Italic)입니다.* (또는 _기울임꼴_)
**이 텍스트는 굵게(Bold) 표시됩니다.** (또는 __굵게__)
***이 텍스트는 굵은 기울임꼴입니다.***
~~이 텍스트에는 취소선이 적용됩니다.~~

---

## 3. 목록 (Lists)

### 3.1. 순서가 없는 목록
별표(`*`), 대시(`-`), 플러스(`+`)를 사용하며, 들여쓰기(Tab 또는 공백 2~4칸)로 하위 목록을 만듭니다.

* 동해물과
  - 백두산이
  + 마르고 닳도록

### 3.2. 순서가 있는 목록
숫자와 마침표(`.`)를 사용합니다. 숫자를 무작위로 적어도 출력 시에는 순서대로 정렬됩니다.

1. 첫 번째 항목
3. 두 번째 항목 (작성은 3으로 했어도 2로 출력됨)
2. 세 번째 항목

---

## 4. 링크 및 이미지 (Links & Images)

### 4.1. 링크 삽입
`[표시할 글자](URL 주소)` 형식으로 작성합니다.
[구글 사이트 바로가기](https://www.google.com)

### 4.2. 이미지 삽입
링크 문법 앞에 느낌표(`!`)를 붙이고, 대괄호 안에 이미지 설명을 적습니다.
![마크다운 로고](https://upload.wikimedia.org/wikipedia/commons/4/48/Markdown-mark.svg)

---

## 5. 코드 (Code)

### 5.1. 인라인 코드
문장 중간에 짧은 코드를 넣을 때는 백틱(`) 1개로 감쌉니다.
`printf("Hello");` 문장은 C언어의 출력문입니다.

### 5.2. 코드 블록
여러 줄의 코드를 작성할 때는 백틱(`) 3개로 감싸고, 첫 줄 백틱 옆에 언어 이름을 적어주면 문법 강조(Highlighting)가 적용됩니다.

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
